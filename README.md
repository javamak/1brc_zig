# 1 Billion Row Challenge - Zig Implementation

## Zig version: 0.15.2

## Build

To build the project with `ReleaseFast` optimizations:


```sh
zig build --release=fast
```

## Run

```sh
time ./zig-out/bin/1brc_zig
```

### Execution time  in i7 13700K processor as below

real    0m2.161s
user    0m45.055s
sys     0m0.683s
